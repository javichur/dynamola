Dynamola

Dynamola, the DynamoDB easy library for Lambda functions. (https://github.com/javichur/dynamola)

Dynamola
Version: 1.0.0
License: MIT
Author: Javier Campos (https://javiercampos.es).
Parameters
tableName (string) nombre de la tabla en DynamoDB.
primaryKeyName (string) nombre de la clave principal de la tabla.
primarySortKeyName (string) nombre de la clave de ordenación de la tabla (opcional).

Example
// constructor, para una tabla con clave principal y clave de ordenación:
let myDb = new Dynamola("nombreMiTablaMensajes", "userId", "fechaHora");
// constructor, para una tabla con solo clave principal:
let myDb = new Dynamola("nombreMiTablaUsuarios", "userId", null);
